üìñ Quick Start

Make a simple POST request to download content from any supported platform:

curl -X POST https://detachbackend-production.up.railway.app/api/download \ -H "Content-Type: application/json" \ -d '{ "url": "https://www.instagram.com/reel/DMw7_HDusQY/", "format": "video", "quality": "best" }'

üìã Request Parameters

url (required): The social media URL you want to download
format (optional): "video" or "audio" - defaults to "video"
quality (optional): "best" or "worst" - defaults to "best"
