Detach API Documentation

The Detach API allows you to extract videos, audio, and metadata from over 10 social media platforms including YouTube, TikTok, Instagram, Pinterest, and more.

Base URL

https://api.detach.dev/v1

Key Features

  • Download videos and audio from 10+ social media platforms
  • Extract rich metadata (titles, descriptions, thumbnails)
  • Multiple format options (MP4, M4A, etc.)
  • Anti-bot detection bypass for reliable extraction
  • Real-time processing with average response time under 2 seconds
  • 99.9% uptime SLA with enterprise-grade infrastructure

Authentication

All API requests require authentication using an API key. Include your API key in the Authorization header of every request.

Authorization Header 
Authorization: Bearer YOUR_API_KEY

Keep Your API Key Secure

Never expose your API key in client-side code or public repositories. Use environment variables or secure configuration management.

Download Content

The main endpoint for downloading content from supported platforms.

POST /v1/download

Request Parameters

Parameter Type Required Description
url string Required The URL of the content to download
format string Optional Download format: "video" (default) or "audio"
quality string Optional Quality preference: "best" (default), "worst", "720p", etc.
webhook_url string Optional Webhook URL for async notifications

Response Fields

Field Type Description
success boolean Whether the download was successful
filename string Generated filename for the downloaded content
fileSize integer File size in bytes
downloadUrl string Temporary download URL (expires in 24h)
platform string Source platform (youtube, tiktok, etc.)
title string Content title/description
uploader string Channel/user name
duration integer Duration in seconds
thumbnail string Thumbnail image URL

Error Handling

The API uses conventional HTTP response codes to indicate success or failure. Error responses include detailed information to help you debug issues.

HTTP Status Codes

Code Status Description
200 Success Download completed successfully
400 Bad Request Invalid request parameters
401 Unauthorized Invalid or missing API key
403 Forbidden Content blocked or private
404 Not Found Content not found or URL invalid
429 Rate Limited Too many requests
500 Server Error Internal server error

JavaScript Examples

Basic Download

Node.js / Browser 
async function downloadVideo(url) {
  try {
    const response = await fetch('https://api.detach.dev/v1/download', {
      method: 'POST',
      headers: {
        'Authorization': 'Bearer YOUR_API_KEY',
        'Content-Type': 'application/json'
      },
      body: JSON.stringify({
        url: url,
        format: 'video',
        quality: 'best'
      })
    });
    
    if (!response.ok) {
      throw new Error(`HTTP error! status: ${response.status}`);
    }
    
    const result = await response.json();
    
    if (result.success) {
      console.log('Download successful!');
      console.log('Title:', result.title);
      console.log('Download URL:', result.downloadUrl);
      return result;
    } else {
      throw new Error(result.message || 'Download failed');
    }
  } catch (error) {
    console.error('Error downloading video:', error);
    throw error;
  }
}

// Usage
downloadVideo('https://youtube.com/watch?v=dQw4w9WgXcQ')
  .then(result => {
    // Handle successful download
    console.log('Downloaded:', result.filename);
  })
  .catch(error => {
    // Handle error
    console.error('Failed to download:', error.message);
  });

Batch Processing

Multiple URLs 
async function downloadMultiple(urls) {
  const results = [];
  
  // Process 3 downloads concurrently to respect rate limits
  for (let i = 0; i < urls.length; i += 3) {
    const batch = urls.slice(i, i + 3);
    const promises = batch.map(url => downloadVideo(url));
    
    try {
      const batchResults = await Promise.allSettled(promises);
      results.push(...batchResults);
      
      // Small delay between batches
      if (i + 3 < urls.length) {
        await new Promise(resolve => setTimeout(resolve, 1000));
      }
    } catch (error) {
      console.error('Batch processing error:', error);
    }
  }
  
  return results;
}

// Usage
const videoUrls = [
  'https://youtube.com/watch?v=dQw4w9WgXcQ',
  'https://tiktok.com/@user/video/123456789',
  'https://instagram.com/p/ABC123/'
];

downloadMultiple(videoUrls)
  .then(results => {
    results.forEach((result, index) => {
      if (result.status === 'fulfilled') {
        console.log(`Download ${index + 1}: Success`);
      } else {
        console.log(`Download ${index + 1}: Failed -`, result.reason.message);
      }
    });
  });

Python Examples

Basic Download

Using requests library 
import requests
import json

class DetachAPI:
    def __init__(self, api_key):
        self.api_key = api_key
        self.base_url = "https://api.detach.dev/v1"
        self.headers = {
            "Authorization": f"Bearer {api_key}",
            "Content-Type": "application/json"
        }
    
    def download(self, url, format_type="video", quality="best"):
        """Download content from a URL"""
        endpoint = f"{self.base_url}/download"
        
        payload = {
            "url": url,
            "format": format_type,
            "quality": quality
        }
        
        try:
            response = requests.post(endpoint, headers=self.headers, json=payload)
            response.raise_for_status()
            
            result = response.json()
            
            if result.get("success"):
                print(f"‚úÖ Download successful!")
                print(f"Title: {result.get('title', 'Unknown')}")
                print(f"Platform: {result.get('platform', 'Unknown')}")
                print(f"File size: {result.get('fileSize', 0):,} bytes")
                print(f"Download URL: {result.get('downloadUrl')}")
                return result
            else:
                print(f"‚ùå Download failed: {result.get('message', 'Unknown error')}")
                return None
                
        except requests.exceptions.RequestException as e:
            print(f"‚ùå Request failed: {str(e)}")
            return None
        except json.JSONDecodeError:
            print("‚ùå Invalid JSON response")
            return None

# Usage
api = DetachAPI("YOUR_API_KEY")

# Download a YouTube video
result = api.download("https://youtube.com/watch?v=dQw4w9WgXcQ")

# Download audio only
audio_result = api.download(
    "https://youtube.com/watch?v=dQw4w9WgXcQ", 
    format_type="audio"
)

Advanced Usage with Error Handling

With retry logic and file download 
import requests
import time
import os
from urllib.parse import urlparse

class DetachAPIClient:
    def __init__(self, api_key, max_retries=3):
        self.api_key = api_key
        self.base_url = "https://api.detach.dev/v1"
        self.max_retries = max_retries
        self.headers = {
            "Authorization": f"Bearer {api_key}",
            "Content-Type": "application/json"
        }
    
    def download_with_retry(self, url, **kwargs):
        """Download with retry logic"""
        for attempt in range(self.max_retries):
            try:
                result = self.download(url, **kwargs)
                if result and result.get("success"):
                    return result
                    
                if attempt < self.max_retries - 1:
                    wait_time = 2 ** attempt  # Exponential backoff
                    print(f"‚è≥ Retrying in {wait_time} seconds... (attempt {attempt + 1})")
                    time.sleep(wait_time)
                    
            except Exception as e:
                if attempt < self.max_retries - 1:
                    print(f"‚ùå Attempt {attempt + 1} failed: {str(e)}")
                    time.sleep(2 ** attempt)
                else:
                    raise e
        
        return None
    
    def download_file(self, download_url, filename):
        """Download the actual file from the download URL"""
        try:
            response = requests.get(download_url, stream=True)
            response.raise_for_status()
            
            # Create downloads directory if it doesn't exist
            os.makedirs("downloads", exist_ok=True)
            filepath = os.path.join("downloads", filename)
            
            with open(filepath, "wb") as f:
                for chunk in response.iter_content(chunk_size=8192):
                    if chunk:
                        f.write(chunk)
            
            print(f"üìÅ File saved to: {filepath}")
            return filepath
            
        except Exception as e:
            print(f"‚ùå Failed to download file: {str(e)}")
            return None

# Usage example
client = DetachAPIClient("YOUR_API_KEY")

urls = [
    "https://youtube.com/watch?v=dQw4w9WgXcQ",
    "https://tiktok.com/@user/video/123456",
    "https://instagram.com/p/ABC123/"
]

for url in urls:
    print(f"\nüé• Processing: {url}")
    
    # Download metadata
    result = client.download_with_retry(url)
    
    if result and result.get("success"):
        # Download the actual file
        download_url = result.get("downloadUrl")
        filename = result.get("filename")
        
        if download_url and filename:
            client.download_file(download_url, filename)
    else:
        print(f"‚ùå Failed to process: {url}")

Rate Limit Headers

Every API response includes rate limit information in the headers:

  • ‚Ä¢ X-RateLimit-Limit: Your rate limit ceiling
